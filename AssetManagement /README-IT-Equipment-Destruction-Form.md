# 🗑️ IT Equipment Destruction Form

## 📝 Description

Application PowerShell avec interface graphique permettant de générer automatiquement un rapport de destruction de matériel informatique au format PDF.

L'outil facilite le suivi des équipements mis au rebut en centralisant les informations d'identification du matériel, la méthode de destruction utilisée et les preuves photographiques avant/après destruction.

## ⚙️ Prérequis

- Windows 10 / Windows 11
- PowerShell 5.1 ou supérieur
- Microsoft Word installé
- Fichiers :
  - logo.png
  - Novares.ico

## 🚀 Utilisation

```powershell
.\IT-Equipment-Destruction-Form.ps1
```

## 📋 Fonctionnalités

### 💻 Informations matériel

- Modèle du PC
- Marque
- Service Tag
- Numéro de série du disque
- Utilisateur associé

### 🔒 Traçabilité

- Date de mise au rebut
- Méthode de destruction
- Méthodes personnalisées

### 📷 Gestion des preuves

- Photo avant destruction
- Photo après destruction
- Aperçu intégré dans l'interface

### 📄 Génération automatique

- Création d'un rapport Word
- Export PDF
- Insertion automatique des photos
- Mise en forme normalisée

## ✅ Cas d'usage

- Gestion du cycle de vie des équipements IT
- Procédure de destruction conforme aux exigences de sécurité
- Conservation des preuves de destruction
- Gestion d'inventaire
- Audits et documentation interne

## 🛠️ Technologies utilisées

- PowerShell
- Windows Forms
- Microsoft Office Interop Word
- COM Automation
- Export PDF
